Linear Regression - Explanation

What is Linear Regression?

Linear Regression is a supervised learning algorithm used to predict a continuous numerical value based on one or more input features. It assumes a linear relationship between the independent variable(s) (features) and the dependent variable (target).

Key Concepts

  • Supervised Learning: The algorithm learns from labeled data (input-output pairs).
  • Continuous Output: Predicts numerical values (e.g., prices, temperatures, grades).
  • Linear Relationship: Fits a straight line (in 2D) or hyperplane (in higher dimensions) through the data.

The Mathematical Foundation

Simple Linear Regression (one feature):

y = mx + b

  • y: Predicted output value
  • x: Input feature
  • m: Slope (how much y changes for each unit of x)
  • b: Intercept (y-value when x = 0)

Multiple Linear Regression:

y = b₀ + b₁x₁ + b₂x₂ + ... + bₙxₙ

  • b₀: Intercept
  • b₁, b₂, ..., bₙ: Coefficients for each feature
  • x₁, x₂, ..., xₙ: Input features

How It Works

  1. Data Preparation: Collect labeled training data with features and target values.
  2. Model Training: Use an optimization algorithm (e.g., Ordinary Least Squares) to find the best-fit line.
  3. Cost Function: Minimize the Mean Squared Error (MSE) - the average of squared differences between predicted and actual values.
  4. Prediction: Use the trained model to predict output for new input data.

Common Use Cases

  • Stock Price Prediction: Predict future prices based on historical data.
  • Real Estate: Estimate house prices based on features like size, location, age.
  • Sales Forecasting: Predict revenue based on marketing spend.
  • Academic Performance: Predict grades based on study hours and attendance.
  • Healthcare: Estimate patient recovery time based on treatment parameters.

Advantages & Disadvantages

Advantages Disadvantages
✓ Simple and interpretable ✗ Assumes linear relationship
✓ Computationally efficient ✗ Sensitive to outliers
✓ Fast to train ✗ Poor with non-linear data
✓ Works well with small datasets ✗ Requires feature scaling

Performance Metrics

  • R² Score (Coefficient of Determination): Measures how well the model explains variance (0 to 1, higher is better).
  • Mean Squared Error (MSE): Average of squared prediction errors (lower is better).
  • Root Mean Squared Error (RMSE): Square root of MSE, in the same units as the target variable.
  • Mean Absolute Error (MAE): Average absolute difference between predictions and actual values.

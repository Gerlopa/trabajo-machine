Quadratic Discriminant Analysis - Explanation

What is QDA?

Quadratic Discriminant Analysis (QDA) is a supervised classification algorithm that models each class with its own Gaussian distribution and covariance. It is used to separate classes when they are not linearly separable.

Key Concepts

  • Supervised Learning: The algorithm learns from labeled data (input-output pairs).
  • Classification: Predicts a discrete class label (e.g., 0 = low risk, 1 = high risk).
  • Quadratic Decision Boundary: The boundary between classes is curved (quadratic).
  • Gaussian Assumption: Each class is assumed to have a multivariate normal distribution.
  • Useful for: Small to medium datasets with nonlinear class separation.

Mathematical Foundation

QDA computes class-specific mean vectors μₖ and covariance matrices Σₖ for each class k, then predicts the class with the highest posterior probability:

δₖ(x) = -0.5 log|Σₖ| - 0.5 (x - μₖ)T Σₖ⁻¹ (x - μₖ) + log πₖ

  • x: Input feature vector
  • μₖ: Mean of class k
  • Σₖ: Covariance matrix of class k
  • πₖ: Prior probability of class k

How It Works

  1. Data Preparation: Collect labeled data with features and target class.
  2. Estimate Parameters: Calculate mean and covariance for each class.
  3. Discriminant Function: Compute δₖ(x) for each class and select the class with maximum δₖ(x).
  4. Prediction: Assign new data points to the class with highest posterior probability.

Common Use Cases

  • Medical Diagnosis: Predict risk categories (e.g., heart disease).
  • Fraud Detection: Classify transactions as fraudulent or not.
  • Marketing: Segment customers based on behavior.
  • Biometrics: Classify images or signals with nonlinear separation.

Advantages & Disadvantages

Advantages Disadvantages
✓ Can handle nonlinear class separation ✗ Requires sufficient data to estimate covariance matrices
✓ Simple probabilistic interpretation ✗ Sensitive to outliers
✓ Works well with small to medium datasets ✗ Assumes Gaussian distribution

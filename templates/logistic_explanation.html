Logistic Regression - Explanation

What is Logistic Regression?

Logistic Regression is a supervised learning algorithm used for binary classification problems (predicting one of two outcomes: Yes/No, 0/1, True/False). Despite its name, it's a classification algorithm, not a regression algorithm. It estimates the probability that an instance belongs to a particular class.

Key Concepts

  • Supervised Learning: The algorithm learns from labeled data with known class outcomes.
  • Binary Classification: Predicts one of two possible outcomes (0 or 1).
  • Probability-Based: Outputs a probability between 0 and 1, then classifies based on a threshold (usually 0.5).
  • Sigmoid Function: Uses an S-shaped curve to map predictions to probabilities.

The Mathematical Foundation

Sigmoid Function:

σ(z) = 1 / (1 + e^(-z))

  • Transforms any value into a probability between 0 and 1.
  • When z approaches +∞, σ(z) → 1
  • When z approaches -∞, σ(z) → 0
  • When z = 0, σ(z) = 0.5

Logistic Regression Equation:

P(y=1|x) = 1 / (1 + e^(-(b₀ + b₁x₁ + b₂x₂ + ... + bₙxₙ)))

  • P(y=1|x): Probability that the outcome is 1 given the features
  • b₀: Intercept term
  • b₁, b₂, ..., bₙ: Coefficients for each feature
  • x₁, x₂, ..., xₙ: Input features

How It Works

  1. Data Preparation: Collect labeled training data with features and binary class labels (0 or 1).
  2. Model Training: Use optimization algorithms (e.g., Gradient Descent) to find coefficients that minimize the loss function.
  3. Cost Function: Minimize Cross-Entropy Loss (Log Loss) - measures difference between predicted probabilities and actual labels.
  4. Prediction: Calculate probability for new samples and classify based on a threshold (typically 0.5).
  5. Classification: If probability ≥ 0.5, predict class 1; otherwise, predict class 0.

Common Use Cases

  • Customer Purchase Prediction: Will a customer buy or not?
  • Disease Diagnosis: Does a patient have a disease or not?
  • Email Spam Detection: Is an email spam or legitimate?
  • Credit Approval: Approve or deny a loan application?
  • College Admission: Will a student be accepted or rejected?
  • Customer Churn: Will a customer stay or leave?
  • Vehicle Insurance: Will a customer file a claim or not?

Logistic Regression vs Linear Regression

Aspect Linear Regression Logistic Regression
Purpose Predict continuous values Predict class/probability
Output Any real number Probability (0 to 1)
Output Function Linear function Sigmoid function
Cost Function Mean Squared Error Cross-Entropy Loss
Problem Type Regression Classification

Advantages & Disadvantages

Advantages Disadvantages
✓ Interpretable coefficients ✗ Assumes linear decision boundary
✓ Outputs probabilities ✗ Limited to binary classification
✓ Fast and efficient ✗ Poor with non-linear patterns
✓ Works with small datasets ✗ Sensitive to outliers
✓ Probabilistic interpretations ✗ Requires feature scaling for optimal performance

Performance Metrics

  • Accuracy: Percentage of correct predictions overall.
  • Precision: Of positive predictions, how many were actually positive?
  • Recall (Sensitivity): Of actual positive cases, how many did we identify?
  • F1-Score: Harmonic mean of Precision and Recall (balances both).
  • ROC-AUC: Area under the Receiver Operating Characteristic curve (measures classification quality).
  • Confusion Matrix: Shows True Positives, True Negatives, False Positives, and False Negatives.

Classification Threshold

By default, logistic regression uses a 0.5 threshold for binary classification:

  • If predicted probability ≥ 0.5 → Predict class 1
  • If predicted probability < 0.5 → Predict class 0

Note: You can adjust this threshold based on the costs of false positives vs false negatives.

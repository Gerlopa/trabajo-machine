Linear Regression - Basic Concepts

What is Linear Regression?

Linear regression is a supervised learning algorithm used to predict a continuous value based on one or more input features.

Mathematical Formula

Y = β₀ + β₁·X

Where:

  • Y = Predicted value (dependent variable)
  • X = Input feature (independent variable)
  • β₀ = Intercept (value when X = 0)
  • β₁ = Slope (change in Y per unit change in X)

Example

Grade = 2.5 × Study Hours + 50

If a student studies 10 hours: Grade = 2.5 × 10 + 50 = 75
